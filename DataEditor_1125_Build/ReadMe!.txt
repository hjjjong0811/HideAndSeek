script_obj.bytes	: 오브젝트용 스크립트

-------------------- 주의사항

**캐릭터여도 정연, 시체, 하빈 등
오브젝트를 발동시켜서 보게되는 스크립트는 obj


**파일 Open시에 Item이나 ComposeItem으로 잘못 Open한 경우
절대 Save하지 말고 프로그램을 껐다 킬것!!!!
	->Save하면 다른클래스로 저장되서 데이터 다날라갈수있음!!

**신경쓴다고썼지만 아주 안정적이진 않으니
중간중간 데이터파일 복사해서 백업해놓는거 추천

=====================사용방법=======================
- 파일열기or생성

0(열기의 경우). 저장될 데이터파일(.bytes)과 editor exe파일은 같은 경로에 위치시킨다
1. 왼쪽 위 필드에 파일이름을 적는다(ex. script_story.bytes)
2. 입력필드 오른쪽의 드랍다운 메뉴를 이용하여 Script를 선택한 후 Open버튼을 누른다

- 파일저장

오픈후 그냥 오른쪽 위의 Save버튼을 누른다

- 데이터 추가

0. 파일열기를 먼저 해야함
1. 오른쪽 위의 Create 버튼을 누른다
2. Key는 필수입력(겹칠 시 추가불가능)
3. Name은 '하빈', '정연', '와인통' 등 캐릭터나 오브젝트 이름
4. 드랍다운 메뉴로 캐릭터 스크립트 일러를 선택가능
5. '1' 이라고 써져있는 필드는 스크립트의 길이이다
	스크립트가 너무 긴경우, 나눠야 예쁜경우는 여러개로 나눌것(예시데이터넣었으니 참고)
6. 만약 액자일러, 지도일러 같이 일러스트가 포함되어야하는 스크립트의 경우
	if want display Picture << 이 필드에 이미지 경로 입력
	editor exe파일과의 상대경로임. 확장자까지 명시할것(.png)
7. OK버튼을 누르면 추가된다. Close는 그냥 닫기다

- 데이터 수정

목록에서 추가한 데이터를 누르면 수정가능함
